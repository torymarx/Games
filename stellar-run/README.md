# Stellar Run — GitHub Pages 배포용

모바일 중심의 우주 생존 게임입니다.

## 포함 파일
- `index.html` — 단일 파일 게임

## 게임 방식
- 5개 레인을 오가며 장애물을 피합니다
- `Pulse` 스킬로 주변 위험을 밀어냅니다
- 파워업을 먹어 실드를 회복합니다
- 4개의 스테이지를 클리어하면 승리합니다
- 최고 점수는 브라우저 로컬 저장소에 저장됩니다

## 조작
- `←` / `→` 또는 `A` / `D` — 레인 이동
- 드래그 / 탭 — 레인 이동
- `Pulse` 버튼 또는 `Space` — 펄스 발동
- `Esc` — 일시정지 / 재개

- 단계별 난이도 상승
- 모바일용 큰 버튼 UI
- 그래픽 효과와 사운드 효과
- 짧고 진지한 첫 번째 메인 게임
