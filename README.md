# Games Dashboard

이 저장소의 메인 진입점입니다. 현재는 첫 번째 게임 **Stellar Run**만 남겨두고, 나머지 초안들은 정리했습니다.

## 구조
- `index.html` — 메인 대시보드
- `stellar-run/index.html` — 모바일 우주 생존 게임
- `.nojekyll` — GitHub Pages의 Jekyll 처리 방지

## 목표
- 모바일에서 재미있게 플레이 가능한 게임
- 단계별 난이도 상승
- 그래픽과 효과음이 있는 완성도 높은 첫 번째 게임

## 추가 방식
새 게임을 더할 때는:
1. `games/<slug>/index.html`로 게임 추가
2. `index.html`의 카드 목록에 추가
3. `README.md`에 목록 업데이트

## 로컬 확인
브라우저에서 `index.html`을 열면 대시보드가 바로 보입니다.
